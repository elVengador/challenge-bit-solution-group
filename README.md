# How to run it

## Option A

Just copy the env variables into `.env` file, I will share through email

## Option B - Set Up your supabase project

- create supabase a new project
- copy the `initial-migration.sql` into your the `Sql Editor`
- copy your `NUXT_PUBLIC_SUPABASE_URL` and `NUXT_PUBLIC_SUPABASE_KEY` into `.env` file
- pnpm install
- pnpm run dev
